## 继承关系

↳ [Path](path)

↳ **Line**

<a name="d3474432"></a>

## 创建方式

```typescript
chart.line();
view.line();
```

<a name="3d0a2df9"></a>

### 参数

• **connectNulls**? : _boolean_

是否连接空值

• **sortable**? : _boolean_

是否对数据进行排序

• **theme**? : _object_

主题配置

• **visible**? : _boolean_

是否可见
