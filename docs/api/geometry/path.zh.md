## 创建方式

```typescript
chart.path();

```

<a name="3d0a2df9"></a>

### 参数

• **connectNulls**? : _boolean_

是否连接空值

• **sortable**? : _boolean_

是否对数据进行排序

• **theme**? : _object_

主题配置

• **visible**? : _boolean_

是否可见
