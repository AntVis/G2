对数据进行指定字段排序。

## 开始使用

```ts
chart
  .data({
    type: 'sort',
    callback: (a, b) => {
      /* your code */
      return 0;
    },
  });
```

## 配置

| 属性 | 描述 | 类型 | 默认值|
| -------------| ----------------------------------------------------------- | -----------------------------| --------------------|
| callback     |  Array.sort 的 comparator，返回 1，0，-1 代表 > = <            | `(a: any, b: any) => number` | `(a, b) => 0`       |
