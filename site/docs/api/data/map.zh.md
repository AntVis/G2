对数据进行过滤。

## 开始使用

```ts
chart
  .data({
    type: 'map',
    callback: (datum) => {
      /* your code */
      return newDatum;
    },
  });
```

## 配置

| 属性 | 描述 | 类型 | 默认值|
| -------------| ----------------------------------------------------------- | -----------------------------| --------------------|
| callback     |  函数，传入当前数据，输出处理后的数据                             | `(datum: any) => any`        | `(d) => d`          |
